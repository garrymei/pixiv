# 粤次元君 - 阶段 5 版本说明 V1

## 一、版本定位

本版本为“粤次元君”阶段 5 测试版 / 审核准备版，目标是确认 MVP 主链路可用，并为提测、提审与后续预上线准备统一口径。

当前版本重点不在“功能铺满”，而在于：

- 核心链路闭环
- 主要 P1 问题修复
- 发布配置与审核准备资料齐套
- 首批内容和风控策略可补位

## 二、当前版本功能范围

### 2.1 已可用功能

| 模块 | 当前可用范围 |
| --- | --- |
| 登录 / 会话 | 可通过当前 mock 登录链路自动建立会话 |
| 首页 | 可浏览推荐内容、正片、日常、招募内容 |
| 帖子 | 可查看详情、点赞、评论、发布帖子 |
| 活动 | 可查看活动资讯 / 官方活动、可报名官方活动、可在“我的活动”查看 |
| 需求 | 可查看需求详情、发布需求、参与需求、查看“我的需求 / 我的参与” |
| 我的 | 可查看资料摘要、统计入口、编辑资料并回显 |

### 2.2 本版本主打闭环

- 首页浏览 -> 帖子详情
- 发布帖子 -> 我的发布 / 公共列表可见
- 帖子详情 -> 点赞 / 评论成功回显
- 活动列表 -> 活动详情 -> 报名成功 -> 我的活动可见
- 需求列表 -> 需求详情 -> 参与成功 -> 我的参与可见
- 我的页 -> 编辑资料 -> 保存成功并回显

## 四、核心使用路径

建议团队统一按以下路径理解本版本：

### 4.1 内容链路

1. 进入首页
2. 浏览帖子卡片
3. 点击进入帖子详情
4. 点赞 / 评论
5. 从发布入口进入发帖页
6. 上传图片并发布成功
7. 回到首页或“我的发布”查看回显

### 4.2 活动链路

1. 进入活动页
2. 切到“官方活动”
3. 打开活动详情
4. 点击报名
5. 在“我的活动”查看已报名记录

### 4.3 需求链路

1. 进入首页招募区或需求详情入口
2. 查看需求详情
3. 点击“感兴趣”
4. 在“我的参与”查看已参与记录
5. 也可从发布入口新建需求并在“我的需求”查看

### 4.4 我的链路

1. 进入“我的”
2. 查看个人信息与统计
3. 进入编辑资料
4. 修改昵称 / 简介 / 城市 / 身份标签
5. 保存并返回查看回显

## 六、已知问题

- 发布作品允许无图提交：已修复
- 发布作品超长内容未限制：已修复
- 发布作品重复提交：已修复
- 发布需求缺少时间 / 地点 / 非法预算类型仍可提交：已修复
