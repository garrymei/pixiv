# 粤次元君 - 提审准备说明 V1

## 一、文档目的

本说明用于团队在提交微信小程序测试版 / 审核版前，统一对外口径，避免临时口头解释不一致。

适用场景：

- 提审前内部评审
- 对接测试同学
- 对接运营 / 产品 / 审核材料整理

## 二、当前版本说明

“粤次元君”当前版本为面向二次元同好社区的 MVP 小程序，核心能力包括：

- 首页内容浏览
- 帖子发布与互动
- 活动资讯与官方活动报名
- 招募需求发布与参与
- 个人资料查看与编辑

当前版本重点是验证核心链路与数据回显，不包含复杂后台、完整审核系统和正式运营配置。

## 三、当前版本功能范围

### 3.1 已可演示功能

- 首页查看内容流
- 点击帖子进入详情
- 发布帖子并查看发布结果
- 查看活动资讯和官方活动
- 官方活动报名并在“我的活动”查看
- 查看需求详情并参与需求
- 发布需求并在“我的需求”查看
- 进入“我的”页面编辑资料并保存

### 3.2 当前不在演示重点内

- 真实微信授权登录流程
- 举报与复杂审核流程
- 完整协议页展示
- 正式线上运营后台
- 完整埋点、监控、告警体系

## 四、审核演示建议

建议审核或演示时优先走以下路径，避免走到未补齐能力：

### 路径 A：内容浏览与发帖

1. 进入首页
2. 浏览推荐内容
3. 点击任一帖子进入详情
4. 返回后进入发布页
5. 选择图片并发布帖子
6. 进入“我的发布”查看结果

### 路径 B：官方活动报名

1. 进入活动页
2. 切换到“官方活动”
3. 进入活动详情
4. 点击“立即报名”
5. 报名成功后进入“我的活动”查看

### 路径 C：需求参与

1. 从首页招募区或相关入口打开需求详情
2. 点击“感兴趣”
3. 进入“我的参与”查看记录

### 路径 D：个人资料编辑

1. 进入“我的”
2. 点击“编辑资料”
3. 修改昵称 / 简介 / 城市 / 身份标签
4. 保存成功后返回查看最新资料

## 五、提审备注建议

提交审核时建议口径：

- 当前版本为兴趣社区 MVP，主要提供作品分享、活动资讯、官方活动报名、招募协作和基础个人中心能力
- 当前版本重点验证基础社区互动与活动 / 需求闭环
- 后续版本将补充内容治理、协议入口、正式运营配置与更完整的社区规则能力

不建议写法：

- 不要写“已支持完整社交生态”
- 不要写“已具备完善审核系统”
- 不要写未实际交付的功能
