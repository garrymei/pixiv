# 粤次元君 - 预上线总检查清单 V1

## 一、目标

在“提测 / 提审 / 预上线”三种决策之间，提供一份统一总清单，帮助团队快速判断当前版本是否具备继续推进资格。

## 二、当前总体判断

### 当前已具备

- 阶段 5 MVP 主链路可用
- 7 条核心业务链路冒烟通过
- P0 问题为 0
- P1 问题已关闭
- 发布配置、风控、种子内容、提审说明文档已初步齐套

### 当前仍未完全具备

- 正式提审条件未全部满足
- 预上线条件未全部满足
- 正式上线条件明确不满足

当前建议结论：

- 可以继续内部提测
- 可以继续整理审核材料
- 不建议直接正式提审或正式上线

- 结论：暂不具备，需补齐配置与审核项后再进入

## 十一、建议推进顺序

1. 先补发布配置：域名、appid、HTTPS、`tabBar` 图标
2. 再补审核基础：隐私政策、用户协议、举报入口、最小下架能力
3. 再补内容准备：Banner、帖子、活动、需求、测试账号数据
4. 最后做真机弱网补测与提审材料收口
