## 六、测试中发现并已关闭的问题

### 6.1 列表接口默认分页参数缺失

- 现象：前端读取列表接口时出现 `Cannot read properties of null (reading 'list')`。
- 影响：帖子、活动、需求列表会导致主链路中断。
- 原因：后端列表 DTO 未给 `page/pageSize` 默认值，前端未传参数时返回结构异常。
- 处理：
  - `backend/src/modules/posts/dto/list-posts.dto.ts`
  - `backend/src/modules/events/dto/list-events.dto.ts`
  - `backend/src/modules/demands/dto/list-demands.dto.ts`
  - 已补充默认值 `page=1`、`pageSize=10`
- 当前状态：已修复并在本报告对应链路中复测通过。
