# 前端页面路由清单 (Page List)

本清单用于指导前端工程师（如 UniApp 或微信小程序原生开发）配置 `pages.json` 或 `app.json`。

## 一、TabBar 页面 (底部导航栏)

| 页面名称 | 建议路径 (`pages/`) | 页面用途 | 核心组件/模块 | 依赖数据对象 | 是否 MVP |
| --- | --- | --- | --- | --- | --- |
| **首页** | `home/index` | 浏览作品/日常 | Tab组件, 瀑布流组件, 帖子卡片 | `Post` 列表 | ✅ 是 |
| **需求** | `market/index` | 寻找协作机会 | 分类筛选栏, 需求卡片 | `Requirement` 列表 | ✅ 是 |
| **发布** | `publish/index` | 发布中转/入口 | 全屏或半屏菜单弹窗 | 无 | ✅ 是 |
| **活动** | `events/index` | 浏览漫展/活动 | 轮播图, 活动状态标签卡片 | `Event` 列表 | ✅ 是 |
| **我的** | `profile/index` | 个人中心 | 用户信息卡片, 列表导航 | `User` 对象 | ✅ 是 |

---

## 二、子页面 (Sub Pages)

| 页面名称 | 建议路径 (`pages/`) | 页面用途 | 核心组件/模块 | 依赖数据对象 | 是否 MVP |
| --- | --- | --- | --- | --- | --- |
| **帖子详情页** | `home/post-detail` | 看图、看文、评论 | 轮播组件, 富文本, 评论区组件 | `Post`, `Comment` 列表 | ✅ 是 |
| **发布图文页** | `publish/post` | 创作作品/日常 | 图片上传组件, 文本域 | 提交构建 `Post` | ✅ 是 |
| **发布需求页** | `publish/requirement` | 发起协作请求 | 表单(时间/地点/角色), 图片上传 | 提交构建 `Requirement` | ✅ 是 |
| **需求详情页** | `market/req-detail` | 查看需求、接单 | 详情展示组件, 底部操作栏 | `Requirement` 对象 | ✅ 是 |
| **活动详情页** | `events/event-detail` | 查看活动介绍 | 富文本/长图展示, 悬浮按钮 | `Event` 对象 | ✅ 是 |
| **活动报名页** | `events/enroll` | 填写报名信息 | 表单(姓名/联系方式) | 提交构建 `EnrollRecord` | ✅ 是 |
| **编辑资料页** | `profile/edit` | 修改个人信息 | 头像上传, 文本输入框 | `User` 对象 | ✅ 是 |
| **我的发布页** | `profile/my-posts` | 管理自己发的帖子 | 列表组件, 删除按钮 | `Post` 列表 | ✅ 是 |
| **我的活动页** | `profile/my-events` | 查看报名的活动 | 列表组件, 状态标签 | `Event` 列表 | ✅ 是 |
| **我的需求页** | `profile/my-reqs` | 查看发布/接单需求 | Tab组件, 需求卡片 | `Requirement` 列表 | ✅ 是 |

## 三、全局组件/弹窗 (不作为独立页面)

- **登录授权弹窗**：在需要鉴权的页面（如点击点赞、点击发布时）弹出，引导用户进行微信授权登录。
- **接单/应征留言弹窗**：在“需求详情页”点击参与时弹出，用于填写简短留言和联系方式。
- **图片预览组件**：全屏查看高清大图，支持双指缩放和保存到相册。
