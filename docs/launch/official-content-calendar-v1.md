# 粤次元君 - 首月官方内容发布节奏 V1

## 一、目标

冷启动阶段不能只等用户自己发内容，前期必须由官方主动提供更新节奏，把首页、活动页和社区氛围先撑起来。

本方案聚焦首月官方内容节奏设计，目标是：

- 让首页持续有新内容
- 让活动页每周都有更新
- 让用户看到平台是“在运营中”，不是静态内容池
- 给社区示范什么样的帖子、活动和招募值得发布

## 二、内容结构原则

首月内容建议围绕 4 类官方内容持续输出：

| 类型 | 作用 | 主要承接页面 |
| --- | --- | --- |
| 活动资讯 | 保证活动页持续更新，提供城市活动信息密度 | 活动页、首页 Banner、首页推荐 |
| 官方活动 | 形成平台组织力和参与理由 | 活动页官方 tab、首页 Banner |
| 社区示范帖 | 教用户“这个平台适合发什么” | 首页推荐、正片、日常、发现页 |
| 品牌 / IP 内容 | 提升平台识别度和记忆点 | 首页推荐、品牌栏目、活动联动帖 |

## 三、首月发布节奏总览

### 3.1 每周建议频次

按照你的要求，建议采用以下周节奏：

| 内容类型 | 每周建议频次 | 首月合计建议 |
| --- | --- | --- |
| 活动资讯 | 2~3 条 | 8~12 条 |
| 官方招募 / 官方活动 | 1~2 条 | 4~6 条 |
| 社区示范帖 | 3~5 条 | 12~20 条 |
| 品牌 / IP 内容 | 1 条 | 4 条 |

### 3.2 每周总体更新量

建议每周总发文量控制在：

- 7~11 条

这样既能保证更新感，也不会因为前期产能不足导致节奏断裂。

## 四、首月每周节奏设计

### 第 1 周：开场周

目标：

- 先把首页内容密度拉起来
- 让活动页一打开就不是空的
- 让用户快速理解平台定位

建议节奏：

| 类型 | 数量 | 内容方向 |
| --- | --- | --- |
| 活动资讯 | 3 条 | 本周漫展资讯、城市活动预告、交通入场提醒 |
| 官方活动 / 官方招募 | 2 条 | 首场同好活动、首批官方招募 |
| 社区示范帖 | 5 条 | 正片示范、日常示范、返图示范、扩列示范、招募示范 |
| 品牌 / IP 内容 | 1 条 | 粤次元君是什么、适合谁来玩 |

重点：

- 第 1 周内容数量可以略高于后续周
- 核心是把首页和活动页初始内容做厚

### 第 2 周：互动周

目标：

- 从“平台有内容”过渡到“平台有参与感”
- 强化活动与招募的互动氛围

建议节奏：

| 类型 | 数量 | 内容方向 |
| --- | --- | --- |
| 活动资讯 | 2~3 条 | 周末活动预告、展会报名汇总、近期返图征集 |
| 官方活动 / 官方招募 | 1~2 条 | 官方主题招募、线下摄影小活动 |
| 社区示范帖 | 4 条 | 参与感内容、同城扩列、活动返图、拍摄准备帖 |
| 品牌 / IP 内容 | 1 条 | 本周主题 / 官方推荐角色 / IP热点话题 |

重点：

- 官方内容不只发“通知”，要发“可参与”的内容
- 让需求、活动、帖子之间形成联动

### 第 3 周：示范周

目标：

- 继续强化平台调性
- 给用户更多可模仿的发帖样板

建议节奏：

| 类型 | 数量 | 内容方向 |
| --- | --- | --- |
| 活动资讯 | 2 条 | 城市资讯整理、活动时间线提醒 |
| 官方活动 / 官方招募 | 1 条 | 节点活动或主题挑战 |
| 社区示范帖 | 3~5 条 | 正片示范、日常穿搭、妆造过程、返图合集 |
| 品牌 / IP 内容 | 1 条 | 官方栏目内容或角色 / 主题专题 |

重点：

- 这一周多做“内容模板示范”
- 让用户看到平台适合发作品、返图、日常、招募

### 第 4 周：稳定周

目标：

- 维持更新节奏，避免前高后低
- 为第二个月沉淀可持续栏目

建议节奏：

| 类型 | 数量 | 内容方向 |
| --- | --- | --- |
| 活动资讯 | 2 条 | 下月活动前瞻、近期看点整理 |
| 官方活动 / 官方招募 | 1 条 | 下月预热活动或长期招募 |
| 社区示范帖 | 3~4 条 | 本月精选、返图回顾、社区优秀内容示范 |
| 品牌 / IP 内容 | 1 条 | 月度总结 / 下月预告 / 品牌主张内容 |

重点：

- 第 4 周要有“月度回顾”和“下月预告”意识
- 方便自然衔接到第二个月的运营节奏

## 五、内容类型拆解建议

### 5.1 活动资讯

建议选题：

- 本周漫展 / 同好活动汇总
- 广州 / 深圳 / 佛山活动看点
- 展会交通、排队、入场提醒
- 报名时间节点提醒
- 返图征集 / 投稿征集

主要用于：

- 活动页“活动资讯”
- 首页 Banner 跳转
- 首页推荐流补充

### 5.2 官方活动 / 官方招募

建议选题：

- 同城同行招募
- 周末摄影外拍活动
- 主题创作挑战
- 节日打卡活动
- 官方线下交流小活动

主要用于：

- 活动页官方活动 tab
- 首页 Banner
- 首页推荐内容中的活动预热帖

### 5.3 社区示范帖

建议覆盖：

- 正片作品示范
- 日常分享示范
- 活动返图示范
- 招募帖示范
- 同城扩列示范

作用：

- 给用户建立内容样板
- 补足首页推荐流密度
- 带动平台内容风格统一

### 5.4 品牌 / IP 内容

建议方向：

- 平台定位介绍
- 每周角色 / 主题栏目
- 热门 IP 专题内容
- 节点活动主题说明
- 官方品牌内容回顾

作用：

- 增强平台辨识度
- 让官方账号不只是发通知
- 为首页增加稳定品牌内容位

## 六、推荐周排期模板

建议每周按下面结构发：

| 星期 | 建议内容 |
| --- | --- |
| 周一 | 活动资讯 |
| 周二 | 社区示范帖 |
| 周三 | 官方活动 / 官方招募 |
| 周四 | 社区示范帖 |
| 周五 | 活动资讯或品牌 / IP 内容 |
| 周六 | 社区示范帖 |
| 周日 | 活动返图 / 周总结 / 预告 |

如果当周产能较低，也建议至少保持：

- 周一有资讯
- 周三有官方动作
- 周五或周末有社区示范内容

## 七、页面承接设计

### 7.1 首页

首页需要持续被以下内容支撑：

- 社区示范帖
- 品牌 / IP 内容
- 官方活动预热
- 活动返图

目标：

- 每次打开首页，都能看到“最近有在更新”

### 7.2 活动页

活动页需要持续被以下内容支撑：

- 每周活动资讯 2~3 条
- 每周官方活动 / 官方招募 1~2 条

目标：

- 活动页不只是挂着 1 条旧活动

## 八、首月建议栏目化设计

为了避免“想到什么发什么”，建议把官方内容做成固定栏目。

推荐栏目：

- 本周活动速报
- 本周同好招募
- 社区示范作品
- 城市漫展看点
- 本周 IP 主题

这样可以显著降低内容规划成本，也更容易形成持续更新节奏。

## 九、上线前最小执行标准

- [ ] 每周活动资讯达到 2~3 条
- [ ] 每周官方活动 / 官方招募达到 1~2 条
- [ ] 每周社区示范帖达到 3~5 条
- [ ] 每周品牌 / IP 内容达到 1 条
- [ ] 首页每周都有新增内容
- [ ] 活动页每周都有新增内容
- [ ] 首月节奏不是堆在第 1 周，而是能连续 4 周维持

## 十、可直接丢给 Trae 的 Prompt

请为“粤次元君”设计首月官方内容发布节奏。

要求：

1. 包含活动资讯、官方活动、社区示范帖、品牌 / IP 相关内容
2. 以每周维度规划
3. 说明每类内容用于哪些页面和模块
4. 输出 `official-content-calendar-v1.md`
5. 目标是支撑冷启动阶段首页和活动页持续更新
