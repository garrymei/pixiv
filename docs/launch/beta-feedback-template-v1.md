# 粤次元君 - 首轮内测用户反馈模板 V1

## 一、使用说明

本模板用于首轮内测用户提交体验反馈，目标是让反馈可统一汇总，而不是变成零散聊天记录。

建议填写方式：

- 每位用户完成试用任务后填写 1 份
- 有问题请尽量附截图或录屏
- 如果没有遇到问题，也请填写整体感受

## 二、基础信息

### 1. 用户昵称 / 称呼

- 示例：小樱 / 摄影阿凯

### 2. 用户类型

- Coser
- 摄影师
- 妆造相关
- 普通同好
- 活动参与型用户
- 其他

### 3. 试用日期

- 示例：2026-04-05

### 4. 试用时长

- 10 分钟以内
- 10~20 分钟
- 20~30 分钟
- 30 分钟以上

## 三、试用任务完成情况

请勾选你本次完成的任务：

- [ ] 浏览首页内容
- [ ] 点进帖子详情

- [ ] 浏览活动页
- [ ] 报名活动
- [ ] 浏览需求详情
- [ ] 参与需求
