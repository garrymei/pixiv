# 粤次元君 - 首轮上线 / 内测复盘会模板 V1

## 一、使用说明

本模板用于首轮上线或首轮内测结束后的项目复盘会，不是做流水账，而是帮助团队在一次会议里快速回答：

- 这轮到底做了什么
- 用户来了多少
- 内容和活动够不够支撑
- 哪条链路最好、哪条链路最差
- 用户最常提什么
- 哪些问题必须先修
- 下一轮做什么 / 不做什么

## 二、会前准备

复盘会前建议准备以下材料：

- 冷启动期核心数据
- 用户反馈汇总
- 高频问题清单
- 本轮发布内容与活动清单
- 下一轮候选需求池

建议会前由负责人提前汇总，不要在会上临时拼数据。
