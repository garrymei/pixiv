# 粤次元君 - 首轮用户行为与问题分析模板 V1

## 一、使用说明

本模板用于首批用户试用或冷启动第一轮数据回收后的复盘，不是写结论用的文章，而是帮助团队快速回答：

- 用户从哪里来
- 用户进来后主要看了什么
- 哪条链路最容易流失
- 发布 / 报名 / 参与哪里卡住了
- 高频问题是什么
- 哪些功能几乎没人用
